# PhD Comprehensive Exam Slides

Browser-based slide deck for Xiaoyi Lei's PhD comprehensive examination.

## Local preview

Serve the repository root with any static web server and open `index.html`.

Keyboard controls:

- Right arrow, Page Down, or Space: next slide
- Left arrow or Page Up: previous slide
- `P`: print

## Project structure

- `index.html`: slide content
- `styles.css`: presentation styling
- `deck.js`: navigation behavior
- `assets/`: figures and visual assets
- `tools/split-deck.mjs`: one-time extraction utility for the original self-contained HTML deck
